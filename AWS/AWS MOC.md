# AWS MOC

Study hub for AWS (Developer Associate–oriented). One folder per service; notes use consistent names.

| Role | Meaning |
|------|---------|
| **Guide** | Main study note for the service |
| **Quiz** | Practice / exam-style questions |
| **Hints** | Exam traps and quick recall |
| **Extra** | Deeper or related material |

## How to study

1. Skim **Foundations** if you need Region / IAM / HA–DR basics.
2. Work services in the order below (guide → quiz → hints).
3. Finish with **Practice** and **Reviews**.

## Suggested order

**Core:** Global Infrastructure → IAM → EC2 → KMS → VPC → S3  
**Compute & APIs:** Lambda → API Gateway  
**Data & messaging:** DynamoDB → SQS → SNS → RDS Aurora  
**Drill:** Practice → Reviews
